# MANO model files

MANO is a licensed external dependency and its model files are not stored in
this repository. Register and download MANO v1.2 from the official MANO site,
then place at least these files here:

```text
mano_v1_2/models/MANO_LEFT.pkl
mano_v1_2/models/MANO_RIGHT.pkl
```

Do not commit or redistribute the downloaded `.pkl` files.
